Fra tema 3 har jeg både trukket fram mitt eget spill, samt gruppeprosjektet. Vi begynte med å lære hvordan man lager elementer og figurer i Illustrator. Deretter ble vi introdusert til java script, og lærte om hvordan man xxx. I gruppeoppgaven var min rolle å designe spillet i Illustrator, lage vår dokumentasjonssite, og finne argumenter for våre valg. Noe av dette ble også gjort i fellesskap.

HVAD HAR JEG LÆRT?

FIGUR-DESIGN: Jeg lærte at det finnes flere prinsipper man kan bruke som virkemidler i design av figurer og former; det er for eksempel forskjell på runde og spisse former. Dette hadde jeg i tankene når jeg lagde mine figurer.
SCENEDESIGN: Vi lærte flere ting om hvordan man skal lage et layot for en scene. Eksempelvis: For å få fram dybde plasserer man gjerne svakere farger bak, og sterkere farger foran.
AKTIVITETS- OG STATE MACHINE DIAGRAM: Jeg fikk en innføring i hvordan man kan lage disse diagrammene, og ikke minst hvordan de kan forenkle prosessen med å kode i java script.
JAVA SCRIPT: I tema 3 ble vi for første gang introdusert til java script. Selv om jeg har en lang vei å gå, vil jeg argumentere for at jeg har begynt å forstå hvordan språket "tenker".

REFLEKSJON

Tema 3 var en utfordrende prosess. I mitt perspektiv var det nokså overveldende å lære om java script, og jeg opplevde at vi gikk meget fort fram. Jeg sitter igjen med behovet for å forstå enda mer av "hvordan språket tenker". Jeg hadde selvsagt mulighet til å øve med enda mer i prosjekt nummer to, men i et gruppeprosjekt er det ofte naturlig at de som har best ferdigheter i noe, også er de som utfører det. Det hadde nok skapt frustrasjon å skulle stå for koding i java script på vegne av de andre gruppemedlemmene.
